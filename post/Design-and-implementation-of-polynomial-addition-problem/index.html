多项式相加问题的设计与实现方法

问题

【问题描述】

一个多项式可以表达为x的各次幂与系数乘积的和，比如：

2x6+3x5+12x3+6x+20

也就是对应数学中的

2x6+3x5+12x3+6x+202x^6+3x^5+12x^3+6x+20 2x6+3x5+12x3+6x+20

现在，你的程序要读入两个多项式，然后输出这两个多项式的和，也就是把对应的幂上的系数相加然后输出。

程序要处理的幂最大为100。

【输入形式】

总共要输入两个多项式，每个多项式的输入格式如下：

每行输入两个数字，第一个表示幂次，第二个表示该幂次的系数，所有的系数都是整数。第一行一定是最高幂，最后一行一定是0次幂。注意：第一行和最后一行之间不一定按照幂次降低顺序排列；如果某个幂次的系数为0，就不出现在输入数据中了；0次幂的系数为0时还是会出现在输入数据中。

【输出形式】

从最高幂开始依次降到0幂，如：

2x6+3x5+12x3-6x+20

注意其中的x是小写字母x，而且所有的符号之间都没有空格，如果某个幂的系数为0则不需要有那项。

【样例输入】

6 2
5 3
3 12
1 6
0 20
6 2 
5 3
2 12
1 6
0 20

【样例输出】

4x6+6x5+12x3+12x2+12x+40

【评分标准】

要求：将多项式中的项设计为Item类，多项式设计为Polyomial类，设计类时要体现多项式和项是一对多的关系。Polyomial类中提供public Polyomial add(Polyomial other)方法完成两个多项式的相加，返回两者和。

思路

  • 多项式相加，也就是相同幂次的系数项相加，按照一定格式输出
  • 面向对象的方法设计，项类（Item）和多项式类（Polyomial）
  • 主要三部分实现：类设计（属性，方法）；控制台输入控制（最高幂次不超过100,0系数不输入）；控制台输出显示（0次幂的处理，加减号处理）。

实现

类的设计

Item类：

实现了Comparable接口，重写比较方法，用于根据系数排序（最后输出用到）；

属性power幂次和coefficient系数；

方法addItem(Item otherItem)用于项与项之间系数相加；

public class Item implements Comparable<Item>{

    private int power; //幂次
    private int coefficient; //系数

    public int getPower() {
        return power;
    }

    public void setPower(int power) {
        this.power = power;
    }

    public int getCoefficient() {
        return coefficient;
    }

    public void setCoefficient(int coefficient) {
        this.coefficient = coefficient;
    }

    public Item(int power, int coefficient) {
        this.power = power;
        this.coefficient = coefficient;
    }

    public Item() {
    }

    @Override
    public String toString() {
        return "Item{" +
                "power=" + power +
                ", coefficient=" + coefficient +
                '}';
    }
    
    public Item addItem(Item otherItem){
        int coeff = this.coefficient + otherItem.coefficient;
        return new Item(this.power,coeff);
    }

    @Override
    public int compareTo(Item otherItem) {
        if (this.power < otherItem.getPower()) {
            return 1;
        } else {
            return -1;
        }
    }
}

Polyomial类

属性items，用List<Item>体现项与多项式之间的多对一的关系；

公共方法add(Polyomial otherPolyomial)用于两个多项式相加；selfAdd()方法多项式自身去重相加；

根据题意

第一行一定是最高幂，最后一行一定是0次幂。注意：第一行和最后一行之间不一定按照幂次降低顺序排列；

可以得知两点

1）单个多项式的系数是杂乱的需要对其系数排序（Item类安排）

2）单个多项式的系数存在重复相同的可能，需要在两个多项式相加之前先对单个进行去重相加，从而增加selfAdd()方法

import java.util.*;

public class Polyomial {

    private List<Item> items;

    public List<Item> getItems() {
        return items;
    }

    public void setItems(List<Item> items) {
        this.items = items;
    }

    public Polyomial(List<Item> items) {
        this.items = items;
    }

    @Override
    public String toString() {
        return "Polyomial{" +
                "items=" + items +
                '}';
    }

    //多项式相加
    public Polyomial add(Polyomial otherPolyomial){
        List<Item> items = this.items;
        List<Item> deduplicatedItems = new ArrayList<Item>(this.items);
        for(int i=0;i<deduplicatedItems.size();i++){
            for(int j=0;j<otherPolyomial.items.size();j++){
                if(deduplicatedItems.get(i).getPower() == otherPolyomial.items.get(j).getPower()){
                    deduplicatedItems.set(i,items.get(i).addItem(otherPolyomial.items.get(j)));
                    otherPolyomial.items.remove(j--);
                }
            }
        }
        deduplicatedItems.addAll(otherPolyomial.items);
        return new Polyomial(deduplicatedItems);
    }

    //自身先去重相加
    public void selfAdd(){
        int size = this.getItems().size();
        for(int i=0;i<size-1;i++){
            for(int j=i+1;j<size;j++){
                if(this.getItems().get(i).getPower() == this.getItems().get(j).getPower()){
                    this.getItems().set(i,this.getItems().get(i).addItem(this.getItems().get(j)));
                    this.getItems().remove(j--);
                }
            }
        }
    }
}

输入这块是我比较头疼的地方，主要是确认哪些地方需要程序控制以及应该怎样控制。仔细审题

程序要处理的幂最大为100。

每行输入两个数字，第一个表示幂次，第二个表示该幂次的系数，所有的系数都是整数。第一行一定是最高幂，最后一行一定是0次幂。注意：第一行和最后一行之间不一定按照幂次降低顺序排列；如果某个幂次的系数为0，就不出现在输入数据中了；0次幂的系数为0时还是会出现在输入数据中。

题意要求每行输入两个数字，输入三个数字该怎么办？

题意要求所有系数应该为整数，那输入的系数为小数怎么办？

题意要求第一行为最高幂，那后面输入比最高幂高的幂了该怎么办？

这些如果交给程序控制也可以，不符合格式要求直接打印提示语句退出。但是我认为以上这些都交给人来输入控制吧，而且吧题目中也没有对应的报错信息，这么多事儿干嘛？

当然也有我认为需要程序控制的地方，比如程序可以处理最大幂为100；不能输入系数为0

确定了程序控制哪些地方，那是输入时控制还是输入完毕的时候控制呢？.......

Scanner scanner = new Scanner(System.in);
List<Item> inputitems = new ArrayList<Item>();
String nextLine = scanner.nextLine();
while (nextLine != null && !nextLine.equals("")) {
    if(Integer.parseInt(nextLine.split(" ")[0]) > 100){
        System.out.println("power over 100 does not support!");
        return;
    }
    if(Integer.parseInt(nextLine.split(" ")[1]) == 0){
        System.out.println("coefficient 0 should not be inputed!");
        return;
    }
    inputitems.add(new Item(Integer.parseInt(nextLine.split(" ")[0]),Integer.parseInt(nextLine.split(" ")[1])));
    nextLine = scanner.nextLine();
}

这里我选择输入时控制，及时提示及时退出；因为每一多项式最后一行幂次必定为0，但是也存在之间也有0次幂时这种可能，因此不能边输入边获取值，必须输入完毕后获取数据，因此要给个输入完毕的指令；这里我设计为不输入按下回车表示两个多项式系数幂次输入完毕，对应代码中while (nextLine != null && !nextLine.equals(""))条件判断；输入的数据存储到inputitems中。

}

踩坑

list中删除元素，循环中需要注意删除元素后list长度发生变化，有可能会导致出错

list用subList方法截取子list，并不会返回新list，而是返回原有list的视图，原有list变化会影响到sublist
