输出出现次数最多的整数及其出现次数

2020-05-15 2 min read # Java

问题

【问题描述】

输入一组无序的整数，编程输出其中出现次数最多的整数及其出现次数。

【输入形式】

先从标准输入读入整数的个数（大于等于1，小于等于100），然后在下一行输入这些整数，各整数之间以一个空格分隔。

【输出形式】

在标准输出上输出出现次数最多的整数及其出现次数，两者以一个空格分隔；若出现次数最多的整数有多个，则按照整数升序分行输出。

【样例输入】

10
0 -50 0 632 5813 -50 9 -50 0 632

【样例输出】

-50 3
0 3

思路

可以边输入边获取值存入数组，也可输入完毕后一并获取整行数据存进数组。

用TreeMap存放key为输入的值，value为该值出现的次数；而且TreeMap的key默认升序的

实现

public static void main(String[] args) {
    Scanner in = new Scanner(System.in);
    int num = in.nextInt();
    int[] data = new int[num];
    int Max = 0;
    Map<Integer,Integer> max = new TreeMap<>();

    for(int i=0; i< num; i++) {
        data[i] = in.nextInt();

        if(!max.containsKey(data[i])) {
            max.put(data[i], 0);
        }else{
            max.put(data[i], max.get(data[i]) + 1);
        }
        //System.out.println(max);
        Max = Max > max.get(data[i]) ? Max : max.get(data[i]);
    }

    Iterator<Integer> iter = max.keySet().iterator();
    while (iter.hasNext()) {
        Integer key = iter.next();
        if(max.get(key) == Max){
            System.out.println(key + " " + max.get(key));
        }
    }
}
