【Angular】Angular常用命令

常用于新建的指令

ng init ：在当前目录创建新的应用程序。
ng new ：创建新的目录，然后在新建的目录中运行 ng init 命令。

因此 ng new 与 ng init 的功能是相似的，只是 ng new 会为我们创建新的目录。

可以简写成：

ng serve：命令会启动开发服务器，监听文件变化，并在修改这些文件时重新构建此应用。
使用–open（或-o）参数可以自动打开浏览器并访问http://localhost:4200/。
